# 🔐 CipherShield - Secure Data Encryption Platform

CipherShield is a web-based encryption system designed to protect sensitive data using strong cryptographic techniques. The platform provides secure encryption and decryption for text, files, and images, along with additional security analysis features.

---

## 🚀 Features

- 🔑 AES-256 Encryption for strong data security
- 🔓 Secure Decryption with integrity verification
- 📊 Key Strength Analysis (Weak / Medium / Strong)
- 📈 Entropy Calculation for randomness measurement
- ⚠️ Suspicious Content Detection (ML-based)
- 🧾 Activity Logging for monitoring user actions
- 🔐 OTP-based Authentication (2FA)
- 📁 File & Image Encryption Support
- 🚨 Email Alerts for suspicious activity

---

## 🛠️ Technologies Used

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Encryption:** AES (PyCryptodome)
- **Authentication:** Flask-Login + PyOTP
- **Security:** Bcrypt (password hashing)
- **Frontend:** HTML, CSS (Neon UI)
- **Machine Learning:** Rule-based classifier
